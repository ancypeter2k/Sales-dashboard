# Sales Dashboard Frontend

React + Redux dashboard UI for the Sales Dashboard project.

## Responsibilities

- Fetch dashboard data from backend API
- Render KPI summary cards
- Render lead status table
- Render trend and distribution charts
- Handle loading, error, and empty states
- Apply date-range filtering (`7` and `30` days)

## Run Locally

```bash
npm install
npm run dev
```

Optional API override:

```bash
export VITE_API_BASE_URL=http://localhost:5001
```

## Scripts

- `npm run dev`
- `npm run lint`
- `npm run build`
- `npm run preview`

For complete project setup, see the root `README.md`.
